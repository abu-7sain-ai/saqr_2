## 1. صفحة توكنات العقول

داشبورد مخصص يبين استهلاك كل نموذج AI:

```
DeepSeek:  X توكن | X$ هذا الشهر
Grok:      X توكن | X$ هذا الشهر
Claude:    X توكن | X$ هذا الشهر
الإجمالي: X$ هذا الشهر
```

**التفاصيل:**
- كل خبير يُسجّل استهلاكه تلقائياً في Supabase
- الأدمن يشوف كل اليوزرات
- اليوزر يشوف استهلاكه فقط
- تنبيه تليجرام لما يقترب من الحد المحدد
- رسم بياني يومي/أسبوعي/شهري

---

## 2. زر تشغيل/إيقاف الأمير والملك

في واجهة اجتماع الخبراء:

```
الأمير (Claude):  🟢 شغّال | 🔴 موقوف
الملك (LSTM):     🟢 شغّال | 🔴 موقوف
```

**القواعد:**
- لازم واحد منهم شغّال على الأقل
- لو الاثنين شغّالين = يتنافسون على Paper Trading
- لو الملك موقوف = الأمير يقود وحده
- لو الأمير موقوف = الملك يقود (بعد ما يثبت نفسه)
- الزر موجود في صفحة اجتماع الخبراء وصفحة الإعدادات

✅ Python يشتغل على السيرفر فقط
✅ Railway Environment Variables
✅ ملف .env لا يُرفع على GitHub أبداً

## 4. اليوزرات وتسجيل الدخول

**أول مرة:**
```
إيميل + كلمة مرور
        ↓
رمز OTP على تليجرام (2FA)
        ↓
دخول ✅
```

**المرة الثانية وما بعدها:**
```
إيميل + كلمة مرور فقط ✅
```

**ملاحظة:**
البيانات الحيوية (بصمة/وجه) تضاف في المرحلة الثانية
لأنها تحتاج دعم خاص على الموبايل.

**أدوار المستخدمين:**
- Admin: كل الصلاحيات
- Owner: يتحكم بموظفيه فقط
- Viewer: يشوف فقط
