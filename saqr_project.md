## 9. نظام الأمير والملك

### الأمير (Claude — نظام عادي):
- يصدر استراتيجيات بناء على اجتماع الخبراء
- رباعي الأدوار: تحليلي + عرّافي + تنبؤي + Utility-Based
- يحسب: ربح محتمل ÷ مخاطرة لكل استراتيجية
- أولوياته: الحارس في المخاطر | الشارتيست في التوقيت | المذيع في الأخبار
- قاعدة صارمة: 3 استراتيجيات فقط مرتبة حسب احتمالية النجاح

### الملك (LSTM via Keras — نظام تنبؤي متطور):
- يصدر استراتيجيات بناء على اجتماع الخبراء + الذاكرة التاريخية
- يُجرَّب كل فترة (شهر، شهرين، ثلاثة)
- متى تفوّق على الأمير → يُعتمد
- زر تفعيل (أخضر) وزر إيقاف (أحمر) في الواجهة

### المسابقة:
```
الأمير يصدر 3 استراتيجيات
الملك يصدر 3 استراتيجيات (لما يكون عنده بيانات كافية)
يشتغلون بالتوازي على Paper Trading
السوق يقرر مين أذكى — مو نحن ✅
```

## 11. نظام الأصدقاء (Paired Bot System)

### القاعدة:
كل مشاة مرتبط بطيار محدد عند الإنشاء:
```
مشاة الأمير 1 ↔ طيار الأمير 1
مشاة الأمير 2 ↔ طيار الأمير 2
مشاة الملك 1 ↔ طيار الملك 1
```
الرابط ثابت عند الإنشاء ولا يتغير.

### تدفق الفلوس الأوتوماتيكي:

**وقت الاستقرار:**
```
المشاة يشتغلون
السيولة الزايدة تنتظر عند الطيارين
أي صفقة تقفل عند المشاة → فلوسها ترجع للمشاة
```

**لما يتوتر السوق:**
```
الطيارون يشتغلون
أي صفقة تقفل عند المشاة → فلوسها لطيارهم الصديق فوراً
```

**لما يرجع مستقر:**
```
أي صفقة تقفل عند الطيارين → فلوسها لمشاتهم الصديق فوراً
```

### القناص:
- محفظة مستقلة خاصة
- يشتغل وقت الاستقرار فقط
- ما يشارك فلوسه مع أحد
