## المرحلة 6 — Backtesting (VectorBT)

### الهدف:
اختبار الاستراتيجيات على 10 سنوات تاريخية.

### الخطوات:
```
1. ثبّت vectorbt + yfinance
2. جلب 10 سنوات بيانات تاريخية
3. طبّق الاستراتيجية مع التكاليف الثلاثة
4. استخرج Sharpe + Drawdown + Win Rate
5. طبّق Walk-Forward:
   - 7 سنوات بناء
   - 3 سنوات اختبار
6. اكتب دالة is_strategy_approved()
7. لو فشل → أرسل تنبيه تليجرام + خيار إعادة الاجتماع
```

### معايير القبول:
```
✅ Sharpe > 1.0
✅ Drawdown < 15%
✅ Win Rate > 50%
✅ ربح 1-3% شهرياً
```

### معيار النجاح:
```
✅ الاستراتيجية تجتاز المعايير
✅ Walk-Forward يشتغل بدون أخطاء
✅ تليجرام ينبه عند الفشل
```
